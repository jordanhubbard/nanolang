# Chapter 14: Data Formats

**Work with JSON and embedded databases.**

This chapter covers data serialization and storage:

- **JSON** - Parse and generate JSON data
- **SQLite** - Embedded SQL database

## When to Use These Modules

**Use JSON when:**
- Working with web APIs
- Configuration files
- Data interchange between systems
- Serializing structured data

**Use SQLite when:**
- Persistent storage needed
- Complex queries required
- Relational data structure
- Embedded database without server
